# Array

## Static Array
A static array is a fixed length container containing n elements indexablefrom the range [0 , n-1]

### Complexity
|  | Static Array | Dynamic Array |
| --- | --- | --- |
| Access | O(1) | O(1) |
| Search | O(n) | O(n) |
| Insertion | NA | O(n) |
| Appending | NA | O(1) |
| Deletion | NA | O(n) |
